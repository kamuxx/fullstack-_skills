# Skill: Frontend Stylist (The Artist)

## Purpose
To translate the "Design" (Skill 13) into pixel-perfect code. It is the master of **CSS3, Tailwind, animations, and responsiveness**. It prevents "Generic Bootstrap" looks and creating premium, fluid interfaces.

## Context Compatibility (User: Lester Rodriguez)
Deepens the Frontend capability:
-   **CSS Mastery:** Vanilla CSS, SASS/LESS.
-   **TailwindCSS:** Configuration, custom plugins, `apply` directives.
-   **Animation:** Micro-interactions (Hover, Focus), Transitions, Keyframes.
-   **Responsive:** Grid/Flexbox layouts that work on Mobile (Ionic experience) and Desktop (Dashboards).

## Inputs
1.  **Design Tokens / Wireframes:** (From Skill 13).
2.  **Component Structure:** (From Skill 04).

## Process
1.  **Framework Configuration:**
    -   *Tailwind:* Customize `tailwind.config.js` with the specialized palette/fonts.
    -   *Bootstrap:* SASS variable overrides (No default blue).
2.  **Layout Strategy:**
    -   Mobile-First approach.
    -   CSS Grid for complex Dashboard Layouts (Sidebar + Header + Content).
3.  **Visual Polish:**
    -   Add **Glassmorphism** (backdrop-filter).
    -   Add **Micro-animations** (transform: scale on click, skeleton loading).

## Outputs
1.  **Global Styles:** `index.css` / `styles.scss` with normalized resets.
2.  **Component Styles:** Tailored classes or Styled Components.
3.  **Animation Snippets:** Reusable CSS/Framer Motion code.
